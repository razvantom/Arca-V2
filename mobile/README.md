# Mobile (React Native Expo)

## Setup
```bash
cd mobile
cp .env.example .env
npm install
npx expo start
```

Rulează backend-ul înainte (default: http://localhost:3001).

Notă: dacă testezi pe telefon, schimbă `EXPO_PUBLIC_API_BASE_URL` către IP-ul PC-ului din rețeaua locală (ex: http://192.168.1.10:3001).
